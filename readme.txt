== Description ==
DS Store Directory (DSSD) adds to your WordPress installation a clean shopping directory, managed by adding/editing/deleting stores from the backend.

= Features =
* Store Directory list view.
* Store Directory grid view (coming soon).
* Store Directory masonry view (coming soon).
* Toggleable Store "Read More" pages.
* Sortable stores (coming soon).
* Filterable stores.
* Mobile friendly.
* Configurable stores options, such as contact information.


== Installation ==
1. Log in to your WordPress administrator account and Navigate to 'Plugins -> Add New'.
2. Input "DS Store Directory" in the top right hand side search bar.
3. Click on the "Install Now" button for DS Store Directory.
4. Click on the "Activate" button for DS Store Directory.

== Description ==
DS Directory adds to your WordPress installation a clean and flexible directory (e.g. Shops, Services, Products, Cars).

= Features =
* Directory list view.
* Directory grid view (coming soon).
* Directory masonry view (coming soon).
* Toggleable "Read More" pages.
* Sortable options (coming soon).
* Filter options.
* Mobile friendly.

== Installation ==
1. Log in to your WordPress administrator account and Navigate to 'Plugins -> Add New'.
2. Input "DS Directory" in the top right hand side search bar.
3. Click on the "Install Now" button for DS Directory.
4. Click on the "Activate" button for DS Directory.
